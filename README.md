# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## Requirements

- Ensure you have Neovim
- Ensure you have [Nerd Fonts](https://www.nerdfonts.com)
- Ensure you have [ripgrep](https://github.com/BurntSushi/ripgrep)
- Ensure you have [fd](https://github.com/sharkdp/fd)
- Ensure you have a C compiler
- Ensure you have `cmake` installed

```bash
winget install BurntSushi.ripgrep.MSVC sharkdp.fd
```
